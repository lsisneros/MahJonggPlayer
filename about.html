Mah Jongg Player

Start

When you click on "Start Play" the main player window opens:

To start play click on the "Start" button.

If your browser is set ot allow pop-ups the four tabs for each hand plus the "Discard" tab will open. If not only the "East" windows opens. In that case you must close the "East" tab and refresh the main window and the buttons will reappear.

You must then click on the buttons for each of the hands plus the "Discard" button. The tiles are shuffled and dealt to each hand - 14 tiles for East and 13 tiles for each of the others. You can sort the tiles by dragging and dropping each tile. You are then ready to begin the Charleston.
