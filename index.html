Mah Jongg Player

The Game

This website allows you to play "solitaire" Mah Jongg. You will play all four positions yourself.

Of course that means that you can see what tiles each position is holding. You may choose to favor one position or another by discarding tiles that a particular position needs or withold those tiles. How realistic you make the game is entirely up to you.

Before You Begin

It is suggested that you familiarize yourself with the information on how the game is implemented by clicking on the various links above.

The web site supports either touch screen or mouse commands and as you will see it uses "drag and drop" and single click (single tap) extensively.
